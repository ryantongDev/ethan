## 🎖️ 荣誉殿堂（Hall）是什么？

在 **荣誉殿堂** 里你可以：

- 🌟 查看已解锁的**徽章**
- 💰 收集**金币**并购买**皮肤**
- 🏆 查看你在每个游戏里的**最高纪录**

每玩一个游戏都会获得金币和成就，继续玩可以解锁更多！

## ⚙️ 设置

### 难度选择
每个游戏都有 **简单 / 普通 / 困难** 三种难度。

在游戏页面的右上角点击难度按钮即可切换。

### 音乐
部分游戏支持背景音乐，点击"音乐"按钮即可开启或关闭。

## 💡 小提示

- 有些游戏需要滑动屏幕来操作，记得先熟悉操作方法
- 每局结束后记得查看分数，看看有没有打破自己的纪录！
- 荣誉殿堂里的奖励需要多玩才能解锁，继续加油！
